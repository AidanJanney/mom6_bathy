mom6_bathy.mom6bathy module¶

class mom6_bathy.mom6bathy.mom6bathy(grid, min_depth)¶

Bases: object

Bathymetry Generator for MOM6 grids (mom6grid).

MOM6 Simpler Models bathymetry constructor.

Parameters

  • grid (mom6grid) – horizontal grid instance for which the bathymetry is to be created.

  • min_depth (float) – Minimum water column depth. Columns with shallow depths are to be masked out.
